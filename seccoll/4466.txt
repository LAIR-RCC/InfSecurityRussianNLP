Отказ в обслуживании в Microsoft .NET Framework Microsoft .NET Framework 3.xMicrosoft .NET Framework 4.x Описание: Уязвимость позволяет удаленному пользователю осуществить DoS-атаку. Уязвимость существует из-за ошибки в спецификации OData. Удаленный пользователь может вызвать отказ в обслуживании сервера. URL производителя: http://www.microsoft.com/ Решение: Для устранения уязвимости установите исправление с сайта производителя. Ссылки: MS13-007: Vulnerability in Open Data Protocol Could Allow Denial of Service (2769327)
