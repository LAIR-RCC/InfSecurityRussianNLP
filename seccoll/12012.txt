Повышение привилегий в Microsoft Windows Microsoft Windows XPMicrosoft Windows 2003Microsoft Windows VistaMicrosoft Windows 2008Microsoft Windows 7 Описание: Обнаруженные уязвимости позволяют локальному пользователю повысить свои привилегии на системе. 1. Уязвимость существует из-за ошибки в win32k.sys при обработке определенных параметров, связанных с файлом раскладки клавиатуры. Локальный пользователь может вызвать повреждение памяти и повысить свои привилегии на системе. 2. Уязвимость существует из-за ошибки обработки некоторых параметров в win32k.sys при создании процедур перехвата вызовов. Локальный пользователь может повысить свои привилегии на системе. URL производителя: www.microsoft.com Решение: Установите исправление с сайта производителя. 10.07.2012 Добавлено уязвимость #2, изменены секции программа и решение.
