Отказ в обслуживании при обработке web форм в JRuby JRuby версии до 1.6.5.1 Описание: Уязвимость позволяет удаленному пользователю вызвать отказ в обслуживании системы. Уязвимость существует из-за ошибки в функции генерации хеша во время обработки или обновления HTTP POST данных. Удаленный пользователь может с помощью специально сформированного HTTP POST запроса потребить все доступные ресурсы процессора на системе и вызвать отказ в обслуживании. URL производителя: jruby.org Решение: Установите последнюю версию 1.6.5.1 с сайта производителя. Ссылки: http://jruby.org/2011/12/27/jruby-1-6-5-1.html http://www.nruns.com/_downloads/advisory28122011.pdf
