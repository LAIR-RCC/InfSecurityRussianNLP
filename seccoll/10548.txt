Компрометация системы в Microsoft Office Microsoft Office 2007Microsoft Office 2010 Описание: Уязвимость позволяет удаленному пользователю выполнить произвольный код на целевой системе. Уязвимость существует из-за ошибки при обработке CGM (Computer Graphics Metafile) файлов. Удаленный пользователь может с помощью специально сформированного CGM файла или документа Microsoft Office со вложенным графическим CGM контентом. URL производителя: http://www.microsoft.com/ Решение: Для устранения уязвимости установите исправления с сайта производителя. Ссылки: MS12-057: Vulnerability in Microsoft Office Could Allow Remote Code Execution (2731879)
