Отказ в обслуживании при обработке web форм в Rack Rack 1.2.0 и более ранние версии Описание: Уязвимость позволяет удаленному пользователю вызвать отказ в обслуживании системы. Уязвимость существует из-за ошибки в функции генерации хеша во время обработки или обновления HTTP POST данных. Удаленный пользователь может с помощью специально сформированного HTTP POST запроса потребить все доступные ресурсы процессора на системе и вызвать отказ в обслуживании. URL производителя: rack.rubyforge.org Решение: Установите исправление из GIT репозитория производителя. Ссылки: http://www.nruns.com/_downloads/advisory28122011.pdf http://www.ocert.org/advisories/ocert-2011-003.html
