Повышение привилегий в Microsoft Windows Microsoft Windows XP Описание: Уязвимость позволяет пользователю повысить привилегии на системе. Уязвимость существует из-за ошибки состояния операции в win32k.sys при обработке определенных попыток создания потоков. Пользователь может повысить привилегии на системе. URL производителя: http://www.microsoft.com/ Решение: Для устранения уязвимости установите исправления с сайта производителя.
