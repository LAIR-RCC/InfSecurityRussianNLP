SQL-инъекция в WordPress Count Per Day Plugin WordPress Count Per Day Plugin 2.17 и более ранние версии Описание: Уязвимость позволяет удаленному пользователю выполнить произвольные SQL команды в базе данных приложения. Уязвимость существует из-за недостаточной обработки входных данных в параметре "month" в сценарии wp-content/plugins/count-per-day/notes.php. Удаленный пользователь может с помощью специально сформированного запроса выполнить произвольные SQL команды в базе данных приложения. Решение: Установите последнюю версию 3.0 с сайта производителя.
