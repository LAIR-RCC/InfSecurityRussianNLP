Повышение привилегий в Microsoft Windows Microsoft Windows 7Microsoft Windows Server 2008 Описание: Уязвимость позволяет локальному пользователю повысить свои привилегии на системе. Уязвимость существует из-за того, что Windows CSRSS не правильно обрабатывает объекты в памяти. Локальный пользователь может повысить свои привилегии на целевой системе и выполнить произвольный код. URL производителя: http://www.microsoft.com/ Решение: Установите исправление с сайта производителя. Ссылки: MS13-019: Vulnerability in Windows Client/Server Run-time Subsystem (CSRSS) Could Allow Elevation of Privilege Журнал изменений: 13.02.2013 Незначительные изменения в описании уязвимости.
