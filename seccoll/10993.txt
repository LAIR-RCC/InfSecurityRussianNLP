Выполнение произвольного кода в Microsoft Works Microsoft Works 9 Описание: Уязвимость позволяет удаленному пользователю выполнить произвольный код на целевой системе. Уязвимость существует из-за ошибки при обработке Word файлов в Microsoft Works. Удаленный пользователь может с помощью специально сформированного Word файла выполнить произвольный код на целевой системе. URL производителя: www.microsoft.com Решение: Установите исправление с сайта производителя. Microsoft Works 9: http://www.microsoft.com/downloads/details.aspx?familyid=7e48cd96-4b91-4f7b-b8a0-2b88131ba51d Ссылки: MS12-065: Vulnerability in Microsoft Works Could Allow Remote Code Execution (2754670)
