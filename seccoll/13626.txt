Компрометация системы в Microsoft Word Microsoft Word 2007 Описание: Уязвимость позволяет удаленному пользователю выполнить произвольный код на целевой системе. Уязвимость существует из-за некорректной обработки .doc файлов. Удаленный пользователь может с помощью специально сформированного .doc файла выполнить произвольный код на целевой системе. URL производителя: http://www.microsoft.com/ Решение: Для устранения уязвимости установите исправление с сайта производителя.
