Переполнение буфера в Caminova DjVu Browser Plug-in Caminova DjVu Browser Plug-in 6.1.4 Build 27351, возможно другие версии. Описание: Уязвимость позволяет удаленному пользователю выполнить произвольный код на целевой системе. Уязвимость существует из-за ошибки в модуле npdjvu.dll при обработке определенных данных в цепочке "Sjbz". Удаленный пользователь может с помощью специально сформированного ".djvu" файла вызвать переполнение динамической памяти и выполнить произвольный код на целевой системе. URL производителя: http://www.caminova.com/en/downloads/download.aspx?id=1 Решение: Установите последнюю версию 6.1.4 Build 27999 с сайта производителя. Ссылки: https://www.caminova.net/ja/downloads/download.aspx?id=1
