Уязвимость форматной строки в Osiris Osiris 4.2.0, возможно более ранние версии. Описание: Уязвимость позволяет удаленному пользователю вызвать отказ в обслуживании или выполнить произвольный код на целевой системе. Уязвимость форматной строки обнаружена в функционале логирования. Удаленный пользователь может с помощью специально сформированной строки аварийно завершить работу приложения или выполнить произвольный код на целевой системе. URL производителя: osiris.shmoo.com Решение: Установите последнюю версию (4.2.1) с сайта производителя. Ссылки: New osiris packages fix arbitrary code execution
