Выполнение произвольного кода в службе печати в Microsoft Windows Microsoft Windows 7Microsoft Windows 2008Microsoft Windows 2008 R2 Описание: Уязвимость позволяет удаленному пользователю выполнить произвольный код на целевой системе. Уязвимость существует из-за ошибки при обработке задач, отправленных на печать, в Microsoft Windows Print Spooler. Удаленный пользователь может с помощью специально сформированной задачи вызвать повреждение памяти и выполнить произвольный код на целевой системе. URL производителя: www.microsoft.com Решение: Установите исправление с сайта производителя. Ссылки: MS13-001: Vulnerability in Windows Print Spooler Components Could Allow Remote Code Execution (2769369)
