SQL-инъекция в The Address Book The Address Book 1.x, 2.x Описание: Уязвимость позволяет удаленному пользователю выполнить произвольные SQL команды в базе данных приложения. Уязвимость существует из-за недостаточной обработки входных данных в параметрах "username" и "password" в процессе аутентификации. Удаленный пользователь может с помощью специально сформированного запроса выполнить произвольные SQL команды в базе данных приложения. URL производителя: tab-2.sourceforge.net Решение: Установите последнюю версию (2.0-rc4) с сайта производителя.
