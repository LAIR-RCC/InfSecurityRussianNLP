Повышение привилегий в VMware ESXi VMware ESX Server 4.1VMware ESX Server 4.0VMware ESX Server 3.5VMware ESX 4.1VMware ESX 4.0VMware ESX 3.5Уязвимость позволяет локальному пользователю повысить свои привилегии на системе Уязвимость существует из-за ошибки при обработке входных и выходных данных портов. Удаленный пользователь может внести изменение в Read-Only Memory виртуальной DOS машины. Примечание: Уязвимость распространяется только на виртуальные машины с операционной системой Windows XP 32-bit, Windows Server 2003 32-bit, и Windows Server 2003 R2 32-bit.
