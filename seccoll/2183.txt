Компрометация системы в Microsoft Internet Explorer Microsoft Internet Explorer 8 Описание: Уязвимость позволяют уделенному пользователю выполнить произвольный код на целевой системе. Уязвимость существует из-за ошибки использования после освобождения в компоненте CTreeNode. Удаленный пользователь может выполнить произвольный код на целевой системе. URL производителя: http://www.microsoft.com/ Решение: Для устранения уязвимости установите исправление с сайта производителя. Ссылки: MS13-021: Cumulative Security Update for Internet Explorer (2809289) Журнал изменений: 12.03.2013 Незначительные изменения в уведомлении.
