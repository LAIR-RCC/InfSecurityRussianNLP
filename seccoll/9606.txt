Выполнение произвольного кода в Apache Struts Apache Struts версии 2.3.5 по 2.3.31 и 2.5 по 2.5.10 Уязвимость позволяет удаленному пользователю выполнить произвольный код на целевой системе. Уязвимость существует из-за недостаточной обработки входных данных в HTTP заголовке Content-Type. Удаленный пользователь может с помощью специально сформированного HTTP POST запроса осуществить инъекцию Java кода и выполнить его на уязвимой системе. Примечание: уязвимость активно эксплуатируется в настоящее время. Установите последнюю версию 2.3.32 или 2.5.10.1 с сайта производителя.
