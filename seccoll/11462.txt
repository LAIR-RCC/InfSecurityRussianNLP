Компрометация системы в Apache Struts Apache Struts версий до 2.3.1.2 Описание: Уязвимость позволяет удаленному пользователю выполнить произвольный код на целевой системе. Уязвимость существует из-за ошибки в классе ParameterInterceptor. Удаленный пользователь может специально сформированного выражения OGNL выполнить произвольный код на целевой системе. URL производителя: http://apache.org/ Решение: Для устранения уязвимости установите продукт версии 2.3.1.2 с сайта производителя. Ссылки: http://struts.apache.org/2.x/docs/s2-009.html http://blog.o0o.nu/2012/01/cve-2011-3923-yet-another-struts2.html
