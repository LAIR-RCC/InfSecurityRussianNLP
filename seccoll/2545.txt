Межсайтовый скриптинг в @Mail @Mail 3.x, 4.x Описание: Обнаруженные уязвимости позволяют удаленному пользователю произвести XSS нападение. 1. Уязвимость существует из-за недостаточной обработки входных данных. Удаленный пользователь может с помощью специально сформированного запроса выполнить произвольный код сценария в браузере жертвы в контексте безопасности уязвимого сайта. 2. Неизвестная ошибка позволяет удаленному пользователю произвести некоторые действия посредством HTTP запросов без проверки подлинности запроса. Удаленный пользователь может произвести CSRFатаку. Подробности не сообщаются. URL производителя: www.atmail.com Решение: Установите последнюю версию 4.61 с сайта производителя.
