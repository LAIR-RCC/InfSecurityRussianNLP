Теги: Visual Studio, Microsoft Первые сведения о Visual Studio 10 были представлены инженером Microsoft Джеффри Шлиммером, который выступил сегодня с докладом на конференции Visual Studio Extensibility Developer Conference. Корпорация Microsoft начала публиковать первые данные о находящейся в разработке новой версии среды Visual Studio 10. Первые сведения о популярной IDE были представлены инженером Microsoft Джеффри Шлиммером, который выступил сегодня с докладом на конференции Visual Studio Extensibility Developer Conference. Шлиммер отметил, что VS10 будет иметь 4 основных приоритета: Опыт - Microsoft намерена сделать VS основным компонентом разработчиков любого направления, опыта и стиля, Заказчики - предстоящая версия будет ориентирована на разработку приложений для малого и среднего бизнеса, Платформа - VS 10 будет сочетать в себе последние платформенные решения Microsoft из сферы Windows-СУБД-.NET, а также Архитектура - в новой версии среды упор будет сделан на создание полного цикла приложений. Представители Microsoft отмечают, что пока VS 10 находится в сравнительно ранней стадии разработки, поэтому большинство текущих моментов будут изменяться и дополняться. Тем не менее, уже сейчас известно, что архитектура самой Visual Studio 10 обещает быть более гибкой и расширяемой, чем нынешний вариант среды для разработчиков. В Visual Studio 10 будет присутствовать новая Windows Presentation Foundation (WPF), платформа Microsoft Extensibility Framework -- MEF, также известная как COM-модель для управляемого кода. Отдельно в Microsoft обещают поработать над новой системой локализации приложений, чтобы создание разных версий одного и того же приложения на разных языках было максимально простым и быстрым. Шлиммер рассказал, что Visual Studio Tools for Applications (VSTA) в 10-й версии можно будет использовать для макросов и других пользовательских систем расширения функциональности. В системы с управляемым кодом VS 10 позволит встраивать больше оснасток. Как и обычно, в новой версии также заявлены новые протоколы для активации, манипуляции, визуализации, синхронизации и управления разработкой и отладкой. VS 10 будет включать обновления в подсистеме MSBuild - написание программ на Visual C должно стать более удобным с точки зрения синтаксиса, а новый подход должен позволит создавать более масштабируемые и производительные приложения. Заявлена и поддержка систем для языков C# и VB. Отдельно в MS говорят о работе в режиме вывода на несколько мониторов и о создании приложений для многоядерных процессоров. Здесь бОльшую часть работы на себя возьмут компиляторы, а разработчикам придется иметь дело с привычными классами Windows Presentation Foundation. Первые бета-версии VS 10 будут иметь ограниченные функциональные возможности многоядерной компиляции, но к финальному релизу, дата которого пока неизвестна, она будет реализована полностью. Не останутся без работы и интернет-сервисы - софтверный гигант намерен внедрить поддержку Live Search везде, где только можно, а за счет поддержки Live ID/Mesh предоставить разработчикам возможность по созданию систем кроссплатформенной авторизации. Также в корпорации не исключают поддержки систем обмена мгновенными сообщениями и разных продуктов конкурентов, таких как IBM Jazz.
