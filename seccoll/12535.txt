Неавторизованное изменение данных в CDS Invenio CDS Invenio 0.92.1, возможно более ранние версии Описание: Уязвимость позволяет удаленному пользователю произвести неавторизованное изменение данных. Уязвимость существует из-за ошибки в механизме email notification alert. Удаленный авторизованный пользователь может с помощью специально сформированного URL удалить персональные почтовые уведомления других пользователей. Для успешной эксплуатации уязвимости злоумышленник должен знать внутренний UID жертвы. URL производителя: cdsware.cern.ch/invenio/ Решение: Установите последнюю версию 0.99.0 с сайта производителя.
