Повышение привилегий в Microsoft .NET Framework Microsoft .NET Framework 4Microsoft .NET Framework 4.5Server Core installation option Описание: Уязвимость позволяет локальному пользователю повысить привилегии на целевой системе. Уязвимость существует из-за некорректной проверки подлинности объектов, используемых при отражении. Локальный пользователь может повысить свои привилегии на целевой системе. URL производителя: http://www.microsoft.com/ Решение: Для устранения уязвимости установите исправление с сайта производителя. Ссылки: MS12-074: Vulnerabilities in .NET Framework Could Allow Remote Code Execution (2745030)
