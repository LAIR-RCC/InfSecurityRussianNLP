Межсайтовый скриптинг в Visual Studio Team Foundation Server Visual Studio Team Foundation Server Описание: Уязвимость позволяет удаленному пользователю произвести XSS нападение. Уязвимость существует из-за недостаточной обработки входных данных. Удаленный пользователь может с помощью специально сформированного запроса выполнить произвольный код сценария в браузере жертвы в контексте безопасности уязвимого сайта. URL производителя: Решение: Установите последнюю версию с сайта производителя.
