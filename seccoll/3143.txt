OS X OS X (Mac OS X) операционная система от Apple. OS X вторая по популярности в мире операционная система. Её рыночная доля в сегменте настольных компьютеров в августе 2012 года составила 7,13 %. OS X значительно отличается от предыдущих версий Mac OS. Основами OS X являются: Подсистема с открытым кодом Darwin (ядро Mach и набор утилит BSD). Среда программирования Core Foundation (Carbon API, Cocoa API и Java API). Графическая среда Aqua (QuickTime, Quartz Extreme и OpenGL). Технологии CoreImage, CoreAudio и CoreData. Ранние версии OS X поддерживали все компьютеры Macintosh (лептопы, десктопы или серверы) на процессорах PowerPC G3, G4 и G5. Более поздние версии перестают поддерживать старое оборудование.
