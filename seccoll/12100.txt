Компрометация системы в VMware vCenter Server VMWare vCenter Server 5.0 до 5.0u3eVMWare vCenter Server 5.1 до 5.1u3bVMWare vCenter Server 5.5 до 5.5u3VMWare vCenter Server 6.0 до 6.0u1 Уязвимость позволяет удаленному пользователю скомпрометировать систему. Уязвимость существует из-за того, что служба JMX RMI не ограничивает регистрацию MBeans. Удаленный пользователь может выполнить произвольный код по протоколу RMI. Примечание: Использование встроенного в ОС Windows межсетевого экрана устраняет проблему. Установите исправление с сайта производителя.
