Множественные уязвимости в Microsoft Internet Explorer Microsoft Internet Explorer 7Microsoft Internet Explorer 8Microsoft Internet Explorer 9 Описание: Обнаруженные уязвимости позволяют удаленному пользователю выполнить произвольный код на целевой системе. 1. С подробным описанием уязвимости можно ознакомиться по адресу: http://www.securitylab.ru/vulnerability/420102.php 2. Уязвимость существует из-за ошибки при подключении к удаленному объекту при обработке определенных HTML разметок. Удаленный пользователь может вызвать повреждение памяти и выполнить произвольный код на целевой системе. 3. Уязвимость существует из-за ошибки при обработке нулевого байта при создании и инициализации строк. Удаленный пользователь может раскрыть содержимое памяти процесса. 4. Уязвимость существует из-за ошибки при подключении к объектам, которые были удалены при обработке VML. Удаленный пользователь может вызвать повреждение памяти и выполнить произвольный код на целевой системе. URL производителя: http://microsoft.com/ Решение: Для устранения уязвимости установите исправления с сайта производителя.
