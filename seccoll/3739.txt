Microsoft анонсировала первые сервис-паки для новейших настольной и серверной ОС Windows 7 и Windows Server 2008 R2. Дату итогового релиза или подробности графика выпуска бета-версий новинок корпорация отказалась назвать. Кроме мелких исправлений и новшеств, наибольший интерес вызывают новые технологии виртуализации, в том числе механизм динамической памяти Dynamic Memory, а также технология RemoteFX, которая позволяет клиентам полноценно использовать мультимедийные приложения, работающие на сервере терминалов. Любопытно, что сроки выпуска SP1 пока не оговариваются. Механизм Dynamic Memory представляет собой дополнение к серверному гипервизору Hyper-V, которое позволяет объединять всю доступную физическую память на сервере в единый массив. Полученный единый массив виртуальной памяти теперь можно динамически перераспределять между виртуальными машинами. Другими словами, при изменении нагрузки виртуальные машины теперь могут получить дополнительный объем памяти без перезагрузки. Технология RemoteFX вызывает еще больше интереса с ее помощью пользователи на тонких клиентах и обычных ПК могут удаленно работать в среде Windows Aero, просматривать полноформатные видеоматериалы высокой четкости, работать с Silverlight-анимацией и трехмерными приложениями. Качество работы при этом будет на уровне самого мощного ПК, хотя на самом деле все задачи будут выполняться на сервере. Технология RemoteFX, основанная на разработках поглощенной в 2008 г. компании Calista Technologies, обеспечит более естественное воспроизведение аудио и видео в виртуальных приложениях, запускаемых с сервера (с использованием технологий Microsoft App-V и Citrix XenApp), а также в удаленных виртуальных машинах, обслуживаемых на мощностях центров обработки данных (с помощью технологий Citrix XenDesktop на базе гипервизоров XenServer или Hyper-V, применяемых в качестве внутреннего гипервизора в центре обработки данных). Представляя технологию RemoteFX, компания Citrix Systems, давний партнер Microsoft, также сообщила, что собирается дополнить возможности терминального доступа в своих продуктах, которые также будут использовать RemoteFX. В частности, обе компании будут совместно развивать технологию HDX для работы с изображением высокой четкости в виртуальных рабочих столах на базе технологии Citrix XenDesktop. Еще одной важной новостью стало изменение требований для технологии Windows XP Mode. Дело в том, что многие предприятия, особенно малы и средние, хотят перейти на платформу Windows 7 Professional, но у них есть приложения, которые требуют использования операционной системы Windows XP. Ранее технология Windows XP Mode для запуска унаследованных приложений в специальной виртуальной машине требовала поддержки аппаратной виртуализации на уровне процессора. С выпуском SP1 это требование устраняется, так что виртуализованные унаследованные приложения теперь можно запускать практически на любой машине, где запускается Windows 7. Само собой, при наличии аппаратной поддержки виртуализации (Intel VT или AMD-V) технология Windows XP Mode будет задействовать аппаратные ресурсы, ускоряя работу виртуализованных приложений. Для конечных пользователей важно было узнать тот факт, что обновление к технологии Windows XP Mode можно загрузить с сайта Microsoft и использовать уже сейчас, не дожидаясь выпуска SP1.
