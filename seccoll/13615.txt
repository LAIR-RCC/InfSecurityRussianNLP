Обход ограничений безопасности в Apache Derby Apache Derby версии до 10.6.1.0 Описание: Уязвимость позволяет удаленному пользователю обойти некоторые ограничения безопасности. Уязвимость существует из-за некорректной трансформации, используемой в функционале аутентификации "BUILTIN", которая приводит уменьшению размера входных данных для SHA-1 при генерации хеша пароля. Удаленный пользователь может быстрее осуществить взлом пароля с путем определения коллизий. URL производителя: http://db.apache.org/derby/ Решение: Обновите продукт до версии 10.6.1.0. Ссылки: http://old.nabble.com/-ANNOUNCE--Apache-Derby-10.6.1.0-released-td28610453.html https://issues.apache.org/jira/browse/DERBY-4483 Журнал изменений: 23.10.2011 Изменено описание уязвимости.
