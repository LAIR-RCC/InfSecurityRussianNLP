Уязвимость при обработке имен файлов в Microsoft Windows Microsoft Windows XPMicrosoft Windows 2003Microsoft Windows VistaMicrosoft Windows 2008Microsoft Windows 7Microsoft Windows 2008 R2 Описание: Уязвимость позволяет удаленному пользователю выполнить произвольный код на целевой системе. Уязвимость существует из-за недостаточной проверки границ данных при обработке имен файлов или каталогов. Удаленный пользователь может с помощью специально сформированного имени файла или папки выполнить произвольный код на целевой системе. URL производителя: www.microsoft.com Решение: Установите исправление с сайта производителя. Ссылки: MS12-081: Vulnerability in Windows File Handling Component Could Allow Remote Code Execution (2758857)
