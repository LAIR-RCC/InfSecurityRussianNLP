Раскрытие данных в реализации TLS протокола в Microsoft Windows Microsoft Windows XPMicrosoft Windows 2003Microsoft Windows VistaMicrosoft Windows 2008Microsoft Windows 7 Описание: Уязвимость позволяет удаленному пользователю получить доступ к важным данным. Уязвимость существует из-за ошибки дизайна в реализации Transport Layer Security (TLS) протокола при использовании симметричных шифров в режиме CBC (например, AES). Удаленный пользователь может расшифровать части HTTPS сессий. URL производителя: www.microsoft.com Решение: Установите исправление с сайта производителя.
