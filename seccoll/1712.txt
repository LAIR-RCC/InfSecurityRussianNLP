Отказ в обслуживании при обработке web форм в Ruby Ruby 1.8.7-p352 и более ранние версии. Описание: Уязвимость позволяет удаленному пользователю вызвать отказ в обслуживании системы. Уязвимость существует из-за ошибки в функции генерации хеша во время обработки или обновления HTTP POST данных. Удаленный пользователь может с помощью специально сформированного HTTP POST запроса потребить все доступные ресурсы процессора на системе и вызвать отказ в обслуживании. URL производителя: www.ruby-lang.org/en/ Решение: Установите последнюю версию 1.8.7-p357 с сайта производителя.
